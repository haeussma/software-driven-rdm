Welcome to sdRDM's documentation!
=================================

sdRDM provide a simple way to create and work with data models, respresenting your data structures. sdRDM is all about the underlying data model of your data, 
representing your data and its relations as a programmatic object model. Thus, your data can be initialised as objects and 
exported to any format.

Get started with sdRDM by running the following command

::

   # Using PyPI
   python -m pip install sdRDM

Or build by source

::

   git clone https://github.com/JR-1991/software-driven-rdm.git
   cd software-driven-rdm
   python3 setup.py install
